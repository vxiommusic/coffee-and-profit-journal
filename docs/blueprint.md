# **App Name**: NovaTrade Journal

## Core Features:

- Automated Trade Logging: Automatically record trade details, including entry/exit prices, timestamps, and instrument, directly from connected brokerage APIs.
- Performance Analytics: Analyze trading performance with automatically generated statistics, including win rate, average profit/loss, and risk-adjusted return. Presents these metrics with interactive data visualizations.
- AI-Powered Pattern Recognition: Employ a tool that scans logged trades for recurring patterns (e.g., chart formations, indicators) and provide alerts about potential opportunities or risks, enhancing decision-making.
- Customizable Risk Management: Set personalized risk parameters (e.g., stop-loss levels, position sizing) to automatically flag trades exceeding risk thresholds.
- Journaling and Annotations: Add contextual notes, screenshots, and thoughts to individual trades or trading sessions to document strategy and emotional state.
- Cloud-Based Secure Storage: Ensure data security and accessibility from multiple devices with encrypted cloud storage of trade data and journal entries.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to convey futuristic technology.
- Background color: Dark gray (#222222) for a sophisticated, tech-focused aesthetic.
- Accent color: Violet (#BE29EC) to create visual interest and highlight important data.
- Font: 'Space Grotesk' sans-serif, giving a clean, digital look; for longer text blocks, use 'Inter' sans-serif.
- Use sleek, minimalist icons to represent trade metrics, alerts, and settings.
- Design a modular layout with interactive charts and data visualizations. Prioritize a clean and data-rich user interface.
- Incorporate subtle animations when data updates or new trades are logged, providing dynamic user feedback.
